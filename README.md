# weatherapp
# 🌦️ Weather App

A modern weather application built to provide real-time weather updates, temperature details, humidity levels, wind speed, and forecast information for any city worldwide.

This project focuses on clean UI, fast weather data fetching, and a smooth user experience using modern frontend technologies and weather APIs.

---

# 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature updates
* 💨 Wind speed information
* 💧 Humidity details
* ☁️ Weather condition display
* 📱 Responsive user interface
* ⚡ Fast API integration
* 🎨 Clean and minimal design

---

# 🛠️ Tech Stack

## Frontend

* HTML
* CSS

# ⚙️ Installation & Setup

## 1. Clone the repository

```bash
git clone https://github.com/sekharchowdary774/weatherapp.git
```

## 2. Navigate to project folder

```bash
cd weatherapp
```

## 3. Open the project

You can directly open `index.html` in your browser.

OR

Run using VS Code Live Server.

---

# 🔑 API Setup

This project uses the OpenWeatherMap API.

## Steps:

1. Create an account at OpenWeatherMap
2. Generate your API key
3. Replace the API key inside your JavaScript file

Example:

```javascript
const apiKey = "YOUR_API_KEY";
```
